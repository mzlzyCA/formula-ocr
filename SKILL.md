# Formula OCR

Use the MinerU tool to recognize and extract mathematical formulas from documents and images.

## Instructions

1. When the user provides an image or document containing math formulas, use the mineru tool to perform OCR and extract the equations.
2. Output formulas in the user's preferred format (LaTeX, MathML, Unicode, plain text).
3. If the mineru tool encounters an error, report it clearly and suggest alternatives (check image quality, ensure file is accessible).
4. Handle edge cases: handwritten formulas, complex nested expressions, low-resolution images, mixed text and math content.
